# ToggleLootSpeciazation
魔兽8.0，5人本根据当前boss自动切换专精

需要打开_ToggleLootSpeciazation.lua改一下拾取专精(后面想做成在游戏里面单击设置的。)


有问题咨询老胡：QQ群597022336
